# SoundShelf

A community music sharing board I built when the idea randomly popped into my head. Had Claude help me conceptualize it and it actually nailed what I was going for â€” a turntable-themed site where anyone can drop a Spotify or YouTube link and it shows up as a vinyl record.

**Live:** [soundshelf.ochiba.dev](https://soundshelf.ochiba.dev)

## Stack

- HTML / CSS / JS (no frameworks)
- FastAPI + SQLite backend
- yt-dlp + ffmpeg for 30-second audio previews
- Web Audio API for real-time visualizers
